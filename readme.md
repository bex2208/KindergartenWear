# Kindergarten Wear Bot

A simple Python Telegram bot that provides daily clothing advice for a 3-year-old based on the weather in **Ubstadt-Weiher**. 

---

## Features

- Fetches hourly weather forecast for Ubstadt-Weiher (8 AM–2 PM) from OpenWeatherMap.
- Generates clothing advice using **Google Gemini** AI.
- Sends a concise recommendation via **Telegram** every morning.
